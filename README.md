# Music Era Discovery using PySpark

This project analyzes over 577,000 songs from 1922 to 2020 to discover how music evolved over time using audio features.

## Tools Used
- Python
- PySpark
- Spark MLlib
- PCA
- K-Means Clustering
- Matplotlib
- Google Colab

## Project Goal
To identify hidden musical patterns across decades using unsupervised learning and large-scale data processing.

## Main Results
- Processed 577K+ songs using PySpark
- Applied feature engineering and clustering on 9 audio features
- Selected K=4 using silhouette score
- Found a major shift in dominant music patterns around the 1970s
